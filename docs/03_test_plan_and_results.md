# 테스트 계획 및 결과 보고서

## 1. 테스트 목적
- 정책 문서 기반 질의응답 정확도 검증
- 환각 방지 및 출처 명시 여부 검증

## 2. 테스트 시나리오
1. "만 39세 서울 거주 예비 창업자가 받을 수 있는 지원금은?"
- 검증 포인트: 나이/지역/예비창업 조건 반영 여부
2. "예비창업패키지와 청년창업사관학교 지원 항목 차이점을 알려줘"
- 검증 포인트: 비교 구조 답변, 근거 출처 표시
3. "업력 3년 제조업 기업의 정책자금 신청 가능 여부는?"
- 검증 포인트: 업력/업종 조건 정합성

## 3. 평가 기준
- 정확성(0~5): 문서 근거와 일치 여부
- 완전성(0~5): 질문 핵심 항목 커버 여부
- 출처 품질(0~5): 파일명/페이지 명시 정확성
- 환각률(0~5): 무근거 생성 빈도(낮을수록 좋음)

## 4. 자동 평가 실행
```bash
python app.py eval
```
